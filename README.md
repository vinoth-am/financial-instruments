## Installation

To run the application node and yarn needs to be installed on the local machine.

Clone the repo.

```sh
git clone https://github.com/vinoth-am/financial-instruments.git
```

Install the dependencies using yarn package manager

```sh
yarn add
```

Run the app

```sh
yarn start
```
